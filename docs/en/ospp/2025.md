# OSPP 2025

**Open Source Promotion Plan (OSPP)** is a long-running summer open-source program initiated and supported by the Institute of Software, Chinese Academy of Sciences. It encourages students to participate in open-source development and maintenance, helps communities cultivate new contributors, and strengthens the open-source software supply chain. See the [OSPP website](https://summer-ospp.ac.cn/) for details.

The AstrBot community participated in this program. The project currently published by the community is listed below.

## Plugin Data Storage Refactor

AstrBot's plugin ecosystem historically lacked a unified storage model. Some plugins persisted data through SharedPreference-like helpers while others wrote JSON directly, which made storage behavior inconsistent and increased maintenance and data-safety risks. This project proposes a more secure and efficient storage mechanism together with a unified plugin data access model, so that plugin storage and retrieval become easier to standardize and maintain.

**Project page**: [Plugin Data Storage Refactor](https://summer-ospp.ac.cn/org/prodetail/253550342?lang=zh&list=pro)

**Difficulty**: Advanced

**Mentor**: [Soulter](https://github.com/Soulter)

**Expected effort**: 210 hours

**Expected deliverables**

1. Design and implement a unified, efficient plugin data storage interface.
2. Replace the old SharedPreference-based storage path with a safer storage mechanism.
3. Add the related technical documentation.

**Technical requirements**

1. Familiarity with Python, JavaScript, and `asyncio`.
2. Familiarity with relational databases such as SQLite.
3. Familiarity with the AstrBot framework and plugin development.
