## [4.26.7] - 2026-07-19

### Added

- Stream Agent statistics to WebChat after every completed LLM request, including independent usage snapshots.
- Added configurable OpenAI-compatible embedding-dimension send modes.

### Changed

- Refined the Dashboard sidebar rail with compact navigation, tooltips, nested-route highlighting, and retained fork-owned shortcuts.
- Made configuration snapshot persistence asynchronous and revision-aware.
- Improved repeated tool-call detection and avoided repeating purpose explanations before every tool call.

### Fixed

- Prefer a configured model context limit over model metadata.
- Remove unsaved provider sources from local Dashboard state without issuing a delete request.
- Preserve versioned Skill names in downloaded archive filenames.
- Return a safe, structured response for unexpected Dashboard API errors.
- Roll back partial knowledge-base uploads reliably.
- Restore code-block copying on insecure HTTP origins and clear command tooltips when the suggestion panel closes.
- Handle concurrent ChatUI follow-up messages and targeted interruption correctly.
- Keep plugin handler binding idempotent.

### Fork Deviations

- This fork intentionally does not include upstream HTML GenUI, TEI rerank, platform-log removal, or Python versions below 3.14 compatibility paths.

## 中文

### 新增

- 每次 LLM 请求完成后向 WebChat 流式推送 Agent 统计信息，并保留独立的用量快照。
- 新增 OpenAI 兼容 Embedding 维度参数发送模式配置。

### 变更

- 优化 Dashboard 侧边栏折叠模式，提供紧凑导航、悬浮提示、嵌套路由高亮，并保留 fork 自有快捷入口。
- 配置快照持久化改为异步且具备版本感知。
- 改进重复工具调用检测，避免在每次工具调用前重复说明用途。

### 修复

- 配置了模型上下文上限时优先使用该值。
- 删除未保存的 Provider Source 时仅清理 Dashboard 本地状态，不再发送删除请求。
- 下载 Skill 时保留名称中的版本号。
- Dashboard API 出现未预期异常时返回安全且结构化的响应。
- 可靠回滚部分失败的知识库上传。
- 恢复非安全 HTTP 环境下的代码块复制，并在命令建议面板关闭时清除残留提示。
- 正确处理并发 ChatUI follow-up 消息及定向中断。
- 保持插件处理器绑定的幂等性。

### Fork 差异

- 本 fork 有意不包含上游 HTML GenUI、TEI rerank、平台日志移除，以及 Python 3.14 以下版本兼容路径。
