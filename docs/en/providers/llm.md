# LLM Providers

You can configure different large language model services in WebUI at `Providers -> + Add Provider`.

> [!TIP]
> If the model service you want is not listed explicitly, check whether that provider offers an OpenAI-compatible API. If it does, you can usually select `OpenAI` in AstrBot and connect it by overriding the API Base URL.

![image](https://files.astrbot.app/docs/source/images/llm/image.png)

![image](https://files.astrbot.app/docs/source/images/llm/image-1.png)

> The corresponding configuration is stored in the `provider` field of `data/cmd_config.json`.
